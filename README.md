<h1>mfetch</h1>
A minimalist fetch written in bash.
<br>

---

# Installation

To install `mfetch` clone the repo by doing:

`$ git clone https://github.com/numpadd/mfetch/`

then run:

`$ sudo make install`

# Usage

In your terminal type:

`$ mfetch`

# Dependencies

* `bash`
* `coreutils`
* `playerctl` - for spotify support
* `dateutils`
* `wmctrl`
